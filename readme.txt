== Description ==

Through jquery I replaced the original links. Open plugins/new-blog-templates-replace-preview-links/new-blog-templates-replace-preview-links.js and update it
with the links you want.

== Installation ==

1. Open plugins/new-blog-templates-replace-preview-links/new-blog-templates-replace-preview-links.js
2. Update it with the links you want:
   That is the model previewButtons.eq(0).attr("data-blog-url","http://remapdemo.webbudesign.com/?marker=2") - they are ordered like in an index, very, very easy!
3. Upload it to plugins folder (/wp-content/plugins/)
4. Activate the plugin
5. It is done. Use it!!!
